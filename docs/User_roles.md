# User Roles

## Super Admin
- Manage all schools
- View subscriptions
- Manage platform settings

## School Admin
- Manage students
- Manage teachers
- Manage classes
- Manage subjects
- Manage examinations

## Teacher
- Enter marks
- Record attendance
- Write report comments

## Parent
- View child results
- Download report cards
- View announcements

## Student
- View results
- Download report cards
- View timetable

## Accountant
- Manage fees
- Record payments
- Generate financial reports
