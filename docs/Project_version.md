# EduReportCard Cloud

## Vision

EduReportCard Cloud is a cloud-based School Management System designed to serve schools across Cameroon.

Our mission is to provide schools with a secure, modern, and intelligent platform for managing students, teachers, examinations, report cards, attendance, fees, and communication.

## Objectives

- Multi-school platform
- Secure login for every school
- AI-powered report cards
- Automatic grading and ranking
- Online subscriptions
- Parent and student portals
- English and French support
- Built to grow across Africa
