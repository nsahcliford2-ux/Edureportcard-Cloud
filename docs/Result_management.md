# Result Management

## Features

- Enter student scores
- Edit scores
- Delete scores
- Continuous Assessment (CA)
- Examination Scores
- Automatic Total Calculation
- Automatic Average Calculation
- Automatic Grade Calculation
- Automatic Position Ranking
- Teacher Remarks
- Principal Remarks
- Lock Results after approval

## Workflow

Teacher
↓
Select Class
↓
Select Subject
↓
Enter Scores
↓
Save
↓
System Calculates
- Total
- Average
- Grade
- Position
↓
Principal Approves
↓
Generate Report Card
