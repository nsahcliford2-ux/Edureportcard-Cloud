# Report Card System

## Features

- Generate report cards automatically
- Display student information
- Display subject scores
- Display grades
- Display average
- Display class position
- Display attendance
- Teacher's remarks
- Principal's remarks
- School stamp and signature
- School logo
- QR Code for verification
- Export to PDF
- Print report card
